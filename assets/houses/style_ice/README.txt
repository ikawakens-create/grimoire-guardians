スプライトシート配置ガイド — style_ice (こおりのいえ)
================================================================

Geminiで生成した縦長スプライトシートを
このフォルダに「spritesheet.png」という名前で置くだけでOK！

■ 配置するファイル:

  spritesheet.png   — 1024×1376px（Special tier: 4セクション）

■ セクション構成（各344px）:

  tower  : Y:0    H:344  → 氷の水晶スパイア
  floor2 : Y:344  H:344  → 2階（ゴシック窓・氷柱）
  floor1 : Y:688  H:344  → 1階（アーチドア）
  garden : Y:1032 H:344  → 雪の庭

■ ファイルサイズ目安: 800KB以下推奨（PNG圧縮後）
