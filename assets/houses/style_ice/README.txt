画像配置ガイド — style_ice (こおりのいえ)
=========================================

■ 必要なファイルは1枚だけ！

  spritesheet.png  — Gemini生成の縦長スプライトシート（512×2064px, 6行）
                     を「そのまま」このフォルダに置くだけでOK。
                     スライス（切り分け）は不要です。

■ CSSが自動的に各行を切り取って表示します:

  tower  (ROW1+ROW2): Y:0    H:688  → 氷の尖塔 + ダイヤ壁
  floor3 (ROW3)     : Y:688  H:344  → 3階壁（フロスト窓）
  floor2 (ROW4)     : Y:1032 H:344  → 2階壁（氷柱窓） ←Geminiのラベル誤記は無視
  floor1 (ROW5)     : Y:1376 H:344  → 1階壁（アーチドア） ←同上
  garden (ROW6)     : Y:1720 H:344  → 雪庭・氷フェンス

■ Geminiから保存するファイル名を必ず「spritesheet.png」にしてください。
