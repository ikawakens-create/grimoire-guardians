スプライトシート配置ガイド — style_flame (ほのおのいえ)
================================================================

Geminiで生成した縦長スプライトシートを
このフォルダに「spritesheet.png」という名前で置くだけでOK！

■ 配置するファイル:

  spritesheet.png   — Gemini生成の縦長1枚画像（512×2064px推奨）

■ スライス不要！

  ゲームコードが自動でCSSを使って各レイヤーを切り出します。
  物理的に画像をスライスする必要はありません。

■ CSS切り出し座標（参考情報）:

  tower  = ROW1+ROW2: Y:0    H:688px  → 屋根・上層壁
  floor3 = ROW3:      Y:688  H:344px  → 3階壁
  floor2 = ROW4:      Y:1032 H:344px  → 2階壁
  floor1 = ROW5:      Y:1376 H:344px  → 1階・玄関
  garden = ROW6:      Y:1720 H:344px  → 庭

■ ファイルサイズ目安: 500KB以下推奨（PNG圧縮後）
