# /morning — 朝のスタートアップ

開発セッション開始時に実行するコマンド。現状把握と今日のタスク確認を行う。

## 実行手順

1. `.claude/checkpoint.md` を読んで前回の引き継ぎ内容を確認する
2. `git log --oneline -5` で最近のコミットを確認する
3. `git status` で未コミットの変更がないか確認する
4. `.claude/progress.md` を読んで全体進捗を確認する
5. 以下の形式で今日のサマリーを出力する

## 出力フォーマット

```
## おはようございます！今日の開発サマリー

### 📌 前回からの引き継ぎ
（checkpoint.md の内容をまとめる）

### 📝 最近のコミット
（git log の内容）

### 🎯 今日おすすめのタスク
1.
2.
3.

### ⚠️ 注意事項
（未解決バグ・ブロッカーがあれば）
```
