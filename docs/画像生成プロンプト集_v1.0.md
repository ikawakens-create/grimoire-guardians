# 画像生成プロンプト集 v1.0 — Grimoire Guardians

> **使い方**
> - 各プロンプトをコピーして画像生成AI（Gemini / DALL-E / Midjourney 等）に貼り付ける
> - 生成後は `assets/npcs/` または `assets/story/` 以下の指定パスに配置する
> - 白背景で生成 → `python3 scripts/remove_bg.py` で透過処理（NPC・ボスのみ）

---

## 共通スタイル指定（全 NPC 画像に適用）

```
STYLE ANCHOR — must apply to ALL NPC images:
- Art style: 2D digital chibi illustration, flat color with soft cel-shading
- Line art: Clean, slightly thick black outlines (3–4px weight)
- Lighting: Soft frontal light, subtle drop shadow under character
- Proportions: Chibi ratio — head is roughly equal to body height (1:1~1:1.5)
- Expressions: Highly expressive, large sparkling eyes, exaggerated emotion
- Pose: Slight 3/4 angle (not perfectly frontal), natural personality pose
- Character frame: Full body, character occupies 70–75% of canvas height
- Canvas: Square format (1:1), character horizontally centered
- ★ CRITICAL — Background: Pure white (#FFFFFF), no shadows on background,
  no gradients, no ground shadow — completely flat white for alpha masking
- Reference feel: Animal Crossing × Pokémon anime × Yo-kai Watch warmth
- Characters must feel ALIVE and LOVABLE, not stiff or generic
- Every character needs ONE iconic "signature detail" that makes them instantly recognizable
- Age-appropriate: target audience is Japanese elementary school children (ages 6–8)
```

## 透過処理の手順

NPC（`assets/npcs/`）とボス（`assets/story/boss/`）は白背景で生成後に透過処理が必要です。

### Web版 claude.ai/code で処理する場合

1. `assets/npcs/` に PNG を配置（git push または Web UIでアップロード）
2. 私に「透過処理して」と伝えるだけ

### ローカルで処理する場合

```bash
pip install "rembg[cpu]" pillow
python3 scripts/remove_bg.py
```

詳細は `scripts/README.md` を参照。
